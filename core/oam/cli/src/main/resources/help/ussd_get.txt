Name
	ussd get

SYNOPSIS
	ussd get

DESCRIPTION
	This command is used to view the details of all USSD paramters.

EXAMPLE
	ussd get
	
	Output:

	noroutingruleconfigerrmssg = Not valid short code. Please dial valid short code
	dialogtimeouterrmssg = Request timedout please try again after sometime
	dialogtimeout = 25000
	servererrmssg = Server error, please try again after sometime

SEE ALSO
	ussd set, ussd set noroutingruleconfigerrmssg, ussd set dialogtimeouterrmssg, 
	ussd set dialogtimeout, ussd set servererrmssg
